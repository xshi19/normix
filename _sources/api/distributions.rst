Distributions
=============

.. contents:: Modules
   :local:
   :depth: 1

Each mixture family (Variance Gamma, Normal-Inverse Gamma, Normal-Inverse
Gaussian, Generalized Hyperbolic) documents four classes from one module: the
joint :math:`(X,Y)` law, the marginal-of-:math:`X` law, a scalar
(:math:`d=1`) convenience wrapper, and the factor-analysis variant.
